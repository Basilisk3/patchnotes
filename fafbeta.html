FAF Beta

Welcome to the fafbeta Patchnotes. The changes listed in this patch note are subject to change.

The changes that are currently part of fafbeta are to evaluate the new balance in actual games.

We have been working hard to maintain this update until we feel the changes are ready for the main game in beta.

This page will be updated as often as possible to show what changes are live on fafbeta

-- The Balance Team

Navy

Cybran Destroyer

GitHub Issue: 4948

We increased range for the Salem before and found it to still be lacking considering the nerf to their frigates. This should help bring their early navy in line with the other factions.

  • Max Range: 80 → 85

Frigate Cost Change

GitHub Issue: 4947

The previous nerf to Cybran frig cost is a bit too much of a nerf considering its decreased range. Similarly, Aeon frigs got a nice buff in both cost and range, this turned out to be too much, so we are removing the cost change for that too.

Aeon

  • Energy Cost: 2800 → 2900
  • Mass Cost: 280 → 290
  • Build Time: 1400 → 1450

Cybran

  • Energy Cost: 2600 → 2500
  • Mass Cost: 260 → 250
  • Build Time: 1300 → 1250

Cruisers

Github Issue:4959

This change will go along with the torp bomber nerf (#4956) which will make sending torps directly into cruisers much less efficient. Instead picking off ships out of range will be favored.

Aeon

  • HP: 3250 → 3850
  • AA Range: 75 → 60

UEF

  • HP: 2500 → 3100
  • AA Range: 75 → 60

Cybran

  • HP: 3000 → 3600
  • AA Range: 75 → 60

Seraphim

  • HP: 2400 → 3000
  • AA Range: 65 → 55

T3 Carriers

GitHub Issue: 4955

This change will allow t3 navy to detect stealth subs at a decent distance.

  • Water Vision Radius: 36 → 100
  • Vision Radius: 85 → 100

Atlantis

GitHub Issue: 4954

These changes should make the Atlantis far more viable. The speed increase will improve its micro ability while the size makes ground firing it less effective.

  • Uniform Scale: 0.1 → 0.09
  • Max Speed: 2.8 → 3.3
  • SelectionSizeX: 1.8 → 1.62
  • SelectionSizeZ: 12.8 → 11.52
  • SizeX: 2.8 → 2.52
  • SizeY: 2.3 → 2.07
  • SizeZ: 20 → 18

Air

Torpedo Bombers

GitHub Issue: 4956

This change along with the cruiser change will make cruisers stronger in head on attacks but allow torps to more easily raid the outskirts of an enemies navy.

  • Damage: 750 → 600

Structures

HARMS

GitHub Issue: 4952

Harms are quite overpowered as is. This change will decrease their total power while keeping them practical to spam up quickly. We remove their stealth ability to make targeting them far less annoying and decrease their health and range to allow destroyers stand a chance against them.

  • Removed Personal Stealth
  • HP: 11000 → 7000
  • Build Time: 4097 → 2050
  • Max Radius: 80 → 70
